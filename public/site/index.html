About the site

This website is built with Eleventy, a static site generator, using VSCode. It is hosted on Neocities and uses deploy-to-neocities github action.

Snails is safe-for-work, mobile-friendly, responsive, and best viewed on firefox browsers. I try my best to keep the website as accessable as possible. If you have any comments, concerns, or complaints regarding this, please reach out to me via email. I love feedback and I'm always willing to improve upon things!
