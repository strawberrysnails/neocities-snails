About the site

This website is built with Eleventy, a static site generator, using VSCode. It is hosted on Neocities and uses deploy-to-neocities github action.

Snails.town is safe-for-work, mobile-friendly, and best viewed on firefox browsers with 1920 x 1080 display. I try my best to keep the website as accessible as possible. If you have any comments, concerns, or complaints regarding this, please reach out to me via email. I love feedback and I'm always willing to improve upon things!

The purpose of this website is a creative outlet for myself and to display my interests. With my lack of traditional social media, I've found a comfortable home on the indie web.

The design of this website is inspired by medieval aesthetics, gothic lolita fashion, and various video games. The plain doll from Bloodborne is my site mascot; She's one of my favorite character designs and a lot of my theme is based off of her color palette.

The code for this site is available for viewing on Github, from which you may take inspiration from.

Archives

Archiving is an important part of the internet. As I continiously update this website, I will always do my best to preserve it's previous forms. Please note that most (if not, ALL) links are broken and these archived pages are not mobile-friendly.
