General Software & Tools

  • Calibre – E-book management.
  • Visual Studio Code – Great offline code editor with lots of plugins/customization.
  • KDE Connect – Control PC from phone and vice versa.
  • Flow Launcher – File search and app launcher for Windows.
  • Listary – Alternative to Flow Launcher
  • AutoHotKey – Create custom hotkeys and shortcuts for Windows.

Image & Design Tools

  • Photopea – Free browser-based Photoshop alternative.
  • Dither it! – Online tool for dithering images.
  • Dither Me This – Advanced image dithering and pixel manipulation.
  • Aseprite – Open source pixel art and animation tool.

Eleventy Plugins

  • external-links – Adds target and rel attributes to external links.
  • markdown-it-anchor – Adds anchors to Markdown headers.
  • markdown-it-attrs – Add custom classes and attributes in Markdown.
  • eleventy-plugin-toc – Easy-config Table of Contents generator.
  • eleventy-plugin-post-stats – Generate blog stats. Example
  • eleventy-fetch – Fetch remote content and cache it locally, useful for pulling in data from APIs or external sources at build time.
  • rss – Adds filters and shortcodes for generating RSS feeds.
  • reading-time – Estimates and displays reading time for blog posts, based on word count and average reading speed.

Web-Hosting
